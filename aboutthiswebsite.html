About this Website

This is an informative website, with the intent to portray information detailing about the New Seven Wonders of the World. I built this website with the aid of Bootstrap, CSS and HTML. This project was completed in two days as a submission to the GitHub Code Innovation Series (February Hackathon). In this website, the images are copyright-free, and I have included the links on where I have obtained information from (if you check the References page). Most of the contents of the website are in my own words. I hope you have fun browsing through this website!
